# My Recipes

Project made with TailwindCSS and Firebase.

## Installation

```bash
npm install
```

## Start

```bash
npm start
```

```bash
@import url('https://fonts.googleapis.com/css2?family=Raleway:wght@100;200;300;400;500;600;700;800&display=swap');

font-family: Raleway, sans-serif;
```
